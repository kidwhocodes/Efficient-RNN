# Documentation Index

This folder holds the experiment workflow and reproducibility notes for the Efficient RNN Project.

## Core workflow
- `docs/experiment_workflow.md`: how to run suites, summarize, and plot results.
- `docs/pruning_methods.md`: strategy names and categories (for configs/CSV).
- `docs/modcog_defaults.md`: Mod-Cog `T` policy to avoid truncated trials.

## Project notes
- `docs/professor_method_study.md`: benchmark checklist for the noise-prune study.
- `docs/pipeline_verification.md`: step-by-step verification of the full pipeline.

Legacy configs/scripts and archived results are stored under `archives/`.
